# Setup Dataviz

This theme uses the [Omeka S Datavis module](https://omeka.org/s/modules/Datavis/) to create data visualizations from Omeka S resources.  

## Installation 
For basic out-of-the-box use of the datavis, follow the [Omeka S Developper Manual instructions for installing](https://omeka.org/s/docs/developer/module_docs/Datavis/).
